# Color Chain Reaction

這是色彩連鎖反應遊戲，大概就是三顆以上的球會有反應就會加分數。

## 遊戲介紹

Color Chain Reaction 是一個策略性的益智遊戲，玩家需要在網格中放置彩色球，當相同顏色的球相鄰3個以上時會產生連鎖反應並獲得分數。遊戲包含主遊戲和三個小遊戲模式。

### 主要功能

- **主遊戲**：經典的彩色球連鎖反應遊戲
- **小遊戲模式**：
  - 色彩記憶遊戲：記憶顏色序列
  - 快速匹配遊戲：60秒速度挑戰
  - 時間挑戰遊戲：5秒內點擊所有勾勾
- **多語言支持**：中文/英文切換
- **主題切換**：亮色/暗色模式
- **最高分記錄**：每個遊戲獨立記錄

## 安裝與運行

### 系統需求
- Node.js 16.0 或以上版本
- npm 或 yarn

### 安裝步驟

1. **複製專案**
   ```bash
   git clone [repository-url]
   cd hw2
   ```

2. **安裝npm**
   ```bash
   npm install
   ```

3. **啟動npm**
   ```bash
   npm start
   ```

4. **打開瀏覽器**
   - 自動打開 http://localhost:3000
   - 如果沒有自動打開，請手動訪問該地址

### 建置生產版本
```bash
npm run build
```

## 遊戲玩法

### 主遊戲
1. 點擊任意列放置彩色球
2. 相同顏色相鄰3個以上會消除
3. 連鎖反應可獲得更高分數
4. 特殊球：彩虹球（萬能）、炸彈球（爆炸範圍）
5. 球會自動掉落，別讓頂部填滿！
6. 挑戰更高分數和等級

### 小遊戲
- **色彩記憶**：無時間限制，記憶顏色序列
- **快速匹配**：60秒內快速匹配顏色
- **時間挑戰**：5秒內點擊所有勾勾，成功後繼續下一輪

- **前端框架**：React 18.2.0
- **程式語言**：TypeScript 4.9.0
- **建置工具**：Create React App
- **狀態管理**：React Hooks
- **樣式**：CSS3

### 專案結構
```
src/
├── components/          # React 組件
│   ├── GameBoard.tsx   # 主遊戲面板
│   ├── MiniGamesMenu.tsx # 小遊戲選單
│   ├── ColorMemoryGame.tsx # 色彩記憶遊戲
│   ├── SpeedMatchGame.tsx # 快速匹配遊戲
│   └── TimeChallengeGame.tsx # 時間挑戰遊戲
├── contexts/           # React Context
│   ├── LanguageContext.tsx # 語言切換
│   └── ThemeContext.tsx # 主題切換
├── hooks/              # 自定義 Hooks
├── types/              # TypeScript 類型定義
├── utils/              # 工具函數
└── App.tsx             # 主應用組件
```
