# 使用守則與方法

## 專案概述

這是一個個人履歷網站，展示我的專業背景、研究經驗、專題作品以及生活剪影。網站採用簡約的設計風格，具備完整的中英雙語支援、主題切換、背景顏色選擇等互動功能。


## 設計理念

### 整體風格
- **簡約**：採用乾淨的設計語言，突出內容本身
- **色調**：使用多種的配色方案供使用者選擇
- **個人化**：融入生活照片和個人興趣，展現人格特質

### 版面設計
- **布局**：左側展示主要內容（經驗、學歷、專案），右側展示技能和聯絡資訊
- **設計**：適配桌面、平板、手機等不同螢幕尺寸
- **視覺**：使用卡片式設計和漸層效果，營造立體感


## 技術特色

### 前端技術
- **HTML**：語義化標籤
- **CSS**：調色、動畫效果
- **TypeScript**：模組化程式設計

### 互動功能
1. **語言切換**：完整的中英雙語支援
2. **主題切換**：深色/淺色模式
3. **背景顏色**：8種漸層背景選擇
4. **照片展示**：支援全螢幕檢視
5. **聯絡表單**：即時得到回饋


## 內容結構

### 主要內容
- **研究經驗**：多樣專題生
- **學歷背景**：電機系
- **專題作品**：智慧型交通號誌控制系統、FPGA數位訊號處理器設計
- **技能專長**：Python, C++, JavaScript, TypeScript, Verilog, EDA工具
- **生活剪影**：分為朋友生活、登山健行、美食專區、輕度旅行


## 檔案結構

```
hw1/
├── index.html                    # 主要HTML檔案
├── styles.css                    # 樣式表
├── personal picture.jpg          # 個人頭像照片
├── README.md                     # 專案說明文件
├── src/main.ts                   # TypeScript原始碼
├── dist/main.js                  # 編譯後的JavaScript
├── some_picture/                 # 照片資源目錄 
│   ├── Friends life/             # 朋友生活照片 
│   ├── mountain/                 # 登山健行照片 
│   ├── Food/                     # 美食照片 
│   └── Travel/                   # 旅行照片 
├── some_pdf/                     # 研究文件 
```


## 執行說明

1. 下載整個 `hw1` 資料夾到本地
2. 雙擊 `index.html` 檔案
3. 網站會自動在瀏覽器中開啟


## 功能操作

- **語言切換**：點擊右上角「🌐 中文」按鈕
- **主題切換**：點擊右上角「主題切換」按鈕
- **背景顏色**：點擊右上角「🎨 背景」按鈕
- **照片檢視**：點擊照片可開啟全螢幕檢視模式
- **聯絡表單**：填寫表單並提交
