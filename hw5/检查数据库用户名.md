# 检查数据库用户名

## ⚠️ 重要区别

您刚才查看的是 **个人资料（Profile）** 页面，这不是数据库用户的账号。

数据库用户的账号在 **Database Access** 页面。

## 🔍 如何查看数据库用户名

### 步骤：

1. 登录 [MongoDB Atlas](https://cloud.mongodb.com/)
2. 在左侧菜单中，点击 **SECURITY** → **Database Access**
   （不是 Profile）
3. 在用户列表中，找到您要使用的用户
4. 查看 **Username** 列，确认用户名是什么
