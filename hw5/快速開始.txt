==========================================
    My-X 專案 - API Keys 快速取得指南
==========================================

📖 詳細指南：請查看 API_KEYS_GUIDE.md 檔案

🚀 最快速開始（只需要 3 個）：

1. NEXTAUTH_SECRET
   ⚠️ 請使用以下命令生成：`openssl rand -base64 32`
   或使用線上工具生成一個隨機字串

2. GitHub OAuth（約 2 分鐘）
   ① 前往：https://github.com/settings/developers
   ② 點擊「New OAuth App」
   ③ 填寫：
      - Application name: My-X
      - Homepage URL: http://localhost:3000
      - Authorization callback URL: http://localhost:3000/api/auth/callback/github
   ④ 複製 Client ID 和 Client Secret

3. MongoDB Atlas（約 5 分鐘）
   ① 前往：https://www.mongodb.com/cloud/atlas
   ② 註冊免費帳號
   ③ 建立免費叢集（M0）
   ④ 建立資料庫使用者
   ⑤ 設定網路存取：0.0.0.0/0
   ⑥ 取得連線字串並填入 DATABASE_URL

==========================================

編輯 .env.local 檔案的方式：

1. 在 Cursor 中左側檔案瀏覽器找到 .env.local
2. 雙擊開啟
3. 填入你取得的 API keys
4. 儲存檔案
