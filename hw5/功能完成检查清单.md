# 功能完成检查清单

## ✅ 注册与登录

- [x] **OAuth 登录**：Google/GitHub（Facebook 已移除）
- [x] **UserID 注册**：注册时输入 userID（3-20字符，字母数字下划线）
- [x] **UserID 登录**：可以使用 userID 登录
- [x] **不同 OAuth providers**：同一个人使用不同 OAuth providers 会注册成不同的 userIDs
- [x] **Session 管理**：登录后给 session，未过期可直接登录

## ✅ 主选单（侧边栏）

- [x] **Home**：回到 X 的 home
- [x] **Profile**：进入个人首页
- [x] **Post**：发表贴文（明亮底色按钮）
- [x] **用户信息**：显示头像、姓名、UserID
- [x] **Logout**：点击用户信息弹出 logout 选项
- [x] **Hover 效果**：其他功能按钮 hover 时微亮 highlight

## ✅ 编辑/浏览个人首页

### 编辑模式（自己的 Profile）
- [x] **姓名**：从 OAuth 取得
- [x] **Number of posts**：显示帖子数量
- [x] **回到 Home 的左箭头**
- [x] **背景图**：可编辑
- [x] **Edit Profile 按钮**：点击后弹出编辑窗口
- [x] **大头贴（Avatar）**：中间对齐背景图底部，可编辑
- [x] **姓名**：再次显示
- [x] **@userID**：显示注册时的 userID
- [x] **Brief description**：可编辑
- [x] **Following/Followers**：显示数量
- [x] **Posts 标签**：显示所有 posts 和 reposts
- [x] **Likes 标签**：显示自己按过爱心的文章（仅自己可见）

### 浏览模式（别人的 Profile）
- [x] **Edit Profile 变成 Follow/Following**：根据关注状态显示
- [x] **Posts**：显示该用户所有 post/repost 的文章
- [x] **没有 Likes 标签**：看不到别人的 likes
- [x] **点击 userID**：在 PO 文中点击 userID，会显示该使用者之个人资料（READ only）

## ✅ 发表文章

- [x] **Post 按钮**：在主选单按下 "Post"，会 pop up modal
- [x] **Post 按钮**：按下右下 Post 则发表文章
- [x] **取消按钮**：按下左上 x 则放弃 PO 文（会跳出小视窗询问是否放弃）
  - [x] **Save**：存成 Draft
  - [x] **Discard**：真的放弃（无法 undo）
- [x] **Drafts 按钮**：显示之前放弃之草稿列表
- [x] **发文规范**：
  - [x] 文章长度为 280 字符限制
  - [x] 链接不管长度皆占用 23 字符
  - [x] #HashTag 不算在字符数里头，且无上限
  - [x] @mention 不算在字符数里头，且无上限
  - [x] 文字输入能够辨识链接并建立 hyperlink

## ✅ 阅读文章

### Home Feed
- [x] **All/Following 标签**：最上面的选项
  - [x] **All**：显示所有文章
  - [x] **Following**：显示所 follow 的人所 post 和 repost 的文章
- [x] **排序**：从最新到最旧
- [x] **@mention 链接**：点击后进入该 mentionSomeone 的个人 profile
- [x] **Inline 发表 post**：最上面可以 inline 发表 post，点击输入文字匡后展开类似 Post modal 的 layout（已实现展开式编辑器）

### 文章显示
- [x] **作者头像**
- [x] **发文时间**：几秒以前、几分钟以前、几小时以前、几天前、几月几日、或几年几月几日
- [x] **内容完整呈现**
- [x] **互动数**：留言数、转发数、按爱心数（从左至右）
- [x] **互动功能**：
  - [x] 点击留言数可以留言
  - [x] 点击转发数可以转发（只支援 repost，不支援 quote）
  - [x] 点击按爱心数可以给爱心（toggling on/off，爱心为 on 时应有颜色上的区别）
- [x] **删除贴文**：如果是自己的发文，在右上方的 "…" 打开选项，可以有 "Delete" 删除文章的选项（Note: re-post 的文章不能删除）

### Recursive Comments
- [x] **点击文章**：中间栏会 "route" 到该篇文章，显示该篇文章以及他所有的留言
- [x] **点击留言**：中间栏会 route 到该则留言，该留言会像是一篇文章一样显示在最上面，底下则为留给该留言的留言
- [x] **继续点选下方留言**：会在 route 进入下一层画面
- [x] **左箭头 + Post**：当点击进入某篇文章或某个留言时，最上方会有一个左箭头 + Post，让您可以点击后回到上一层文章列表/文章/留言
